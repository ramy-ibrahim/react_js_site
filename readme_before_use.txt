The requirements
- Bootstrap is required, please install Bootstrap in your project folder using the following commands.
for Windows:
npm install bootstrap

for Linux or macOS
sudo npm install bootstrap 

- Router is required, install the "Router" in your project folder using the following command.

npm install react-router-dom@latest

Also, extract the "fontawesome-free-6.2.0-web.zip" file into the "src" folder.
